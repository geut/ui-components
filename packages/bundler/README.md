# Usage
<!-- usage -->
```sh-session
$ npm install -g bundler
$ bundler COMMAND
running command...
$ bundler (-v|--version|version)
bundler/0.0.0 darwin-x64 node-v12.3.1
$ bundler --help [COMMAND]
USAGE
  $ bundler COMMAND
...
```
